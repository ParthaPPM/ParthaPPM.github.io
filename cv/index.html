Question Answering System in Bengali Using Semantic Search

Question Answering system (QA) is a computer program that takes question as input and retrieves answer using Information Retrieval (IR). This work presents the design, implementation and evaluation of QA system based on semantic analysis using Bengali WordNet. The data sets are prepared from the Bengali text corpus which is developed under the TDIL (Technology Development for Indian Languages) project of the Government of India.

A Knowledge based Approach to Analyze the Sentiment of Online Review

The online reviews about several entities are collected from the internet (eg. Amazon, Flipkart) using a crawler program. The sentiment of individual review is resolved using two algorithms. First one is the Lexical Similarity based approach and second algorithm is based on Context Expansion strategy.
